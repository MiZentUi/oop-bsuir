# Orders system

## Структура и задание

Система поделена на следующие модули:
- `infrastructure`: Имитация работы с БД и внешними сервисами.
- `models`: Модели данных системы.
- `staff`: Система управления персоналом склада.
- `Main.java`: Точка входа в приложение.
- `OrderProcessor.java`: Основная бизнес-логика.
