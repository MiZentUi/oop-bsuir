# Orders system

## Структура и задание

Система поделена на следующие модули:

*   `infrastructure.kt` : Имитация работы с БД и внешними сервисами.
*   `Main.kt` : Точка входа в приложение.
*   `models.kt` : Модели данных системы.
*   `processor.kt` : Основная бизнес-логика.
*   `Staff.kt` : Система управления персоналом склада.

Почти в каждом модуле есть архитектурные ошибки, которые нужно исправить.
