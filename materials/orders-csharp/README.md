# Orders system

## Структура и задание

Система поделена на следующие модули:
- `Infrastructure.cs`: Имитация работы с БД и внешними сервисами.
- `Program.cs`: Точка входа в приложение.
- `Models.cs`: Модели данных системы.
- `Processor.cs`: Основная бизнес-логика.
- `Staff.cs`: Система управления персоналом склада.

Почти в каждом модуле есть архитектурные ошибки, которые нужно исправить.



## Использование
Запуск системы:
```
dotnet run
```
